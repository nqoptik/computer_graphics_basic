# Circle drawing algorithms in computer graphics

## Python code
### Prerequisites
Install pyglet:
```
pip install pyglet
pip3 install pyglet
```
### Run the code
```
cd scripts
python draw_circle.py
python3 draw_circle.py
```

## C++ code
### Prerequisites
Install OpenGL:
```
sudo apt-get install freeglut3-dev
```

### Build the code
```
mkdir build
cd build
cmake ..
make
```

### Run the code
```
./draw_circle
```
