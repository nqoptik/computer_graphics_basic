# Line drawing algorithms in computer graphics

## Python code
### Prerequisites
Install pyglet:
```
pip install pyglet
pip3 install pyglet
```
### Run the code
```
cd scripts
python draw_line.py
python3 draw_line.py
```

## C++ code
### Prerequisites
Install OpenGL:
```
sudo apt-get install freeglut3-dev
```

### Build the code
```
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make
```

### Run the code
```
./draw_line
```
